# Functions and Nested Loops

This directory contains tasks related to functions and nested loops in C programming.

## Tasks
- **0-putchar.c**: Prints `_putchar` followed by a new line.

## Requirements
- Compiled on **Ubuntu 20.04 LTS** using **GCC** with `-Wall -Werror -Wextra -pedantic -std=gnu89`
- Code follows **Betty coding style**
- No use of **standard library functions**
- **`_putchar`** function is allowed
